micro:bit BLE Clock

This project makes the digital clock. The key clock feature is the time synchronization. In general, the digital clock has the button to synchronize the current time. You should push this button many times to set the current time, but this is a boaring task. micro:bit has the BLE feature. So, this project makes use of BLE to synchroniza the current time.

micro:bit has 5x5 LED which can be the clock display, but it is painful to identify the time shown in 5x5 LED. I connected the tiny character LCD, AQM0802. This is controlled vi I2C.

I also connected the speaker. micro:bit is easy to make the sound. The micro:bit BLE Clock sounds the time signal every minute. Below is the schematic.
